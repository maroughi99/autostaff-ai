üîë Get Your AutoStaffAI User ID

Follow any of these methods to get your user ID for testing:

Method 1: From Browser Console (Easiest)

Go to https://autostaffai.com/dashboard
Sign in if you're not already
Press F12 to open Developer Tools
Click the Console tab
Paste this command and press Enter:
fetch('/api/auth/me').then(r => r.json()).then(d => console.log('YOUR USER ID:', d.id || d.clerkId))
Look for "YOUR USER ID:" in the console output

Method 2: From URL (If Visible)

Go to any page in your dashboard
Check the URL - sometimes user ID appears like:
https://autostaffai.com/dashboard/settings?userId=user_2xxxxx

Method 3: From Network Tab

Go to https://autostaffai.com/dashboard
Press F12 ‚Üí Click Network tab
Refresh the page
Look for API requests (like /api/leads or /api/messages)
Click on one and check the Query Params - look for userId

Method 4: From Cookies

Press F12 ‚Üí Go to Application tab
Click Cookies ‚Üí https://autostaffai.com
Look for a cookie named __session or similar
The value contains your session data (though you need the user ID, not session)

üìù Once You Have Your User ID:

Save it to .env.test.local file:

TEST_USER_ID=user_2xxxxxxxxxxxxx

Or use it directly:

$env:TEST_USER_ID="user_2xxxxx"; node test-production.js

üí° Tip: Your user ID usually starts with user_2 followed by random characters.
